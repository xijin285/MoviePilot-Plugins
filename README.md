# 🎬 MoviePilot-Plugins

MoviePilot第三方插件库，提供了一系列实用的插件来增强MoviePilot的功能。

> ⚠️ 注意：本插件库为个人维护，代码结构参考了其他开源项目。推荐优先使用[官方插件库](https://github.com/jxxghp/MoviePilot-Plugins)。

## 📖 使用说明

**本仓库为第三方插件库，需在MoviePilot中添加仓库地址使用**

1. 在MoviePilot的插件商店页面，点击"添加第三方仓库"
2. 添加本仓库地址：`https://github.com/xijin285/MoviePilot-Plugins`
3. 添加成功后，在插件列表中找到需要的插件
4. 安装并启用插件
5. 根据下方说明配置插件参数

## 📦 插件列表

### 🛡️ 爱快路由备份助手 (IkuaiRouterBackup)
-   **版本**: `v1.1.1`
-   **作者**: [@jinxi](https://github.com/xijin285)
-   **简介**: 自动备份您的爱快(iKuai)路由器配置，管理备份文件，为您的网络设置提供一道额外的安全保障。
-   **主要功能**:
    -   ⏰ **定时任务**: 自动执行备份，无需人工干预。
    -   💾 **配置备份**: 完整备份路由器当前配置。
    -   📂 **备份管理**:
        -   自定义备份文件存储路径。
        -   设置备份文件保留数量，自动清理旧备份。
    -   🔔 **结果通知**:
        -   支持多种美观通知样式选择。
        -   清晰反馈备份成功或失败状态。
    -   ⚡ **即时执行**: 支持通过UI触发"立即运行一次"备份。
    -   📊 **历史记录**: 在插件页面查看详细的备份历史。
    -   🔄 **智能重试**: 登录和备份过程中的自动重试机制。
-   **最新更新 (`v1.1.1`)**:
    -   修正了在插件设置界面中，通知样式选择(`VSelect`组件)选项显示为 `[object Object]` 的问题，确保选项文本正常展示。
-   **历史更新**:
    -   `v1.1.0`:
        -   🎨 通知功能全面美化，引入多种可选通知样式(简约星线、方块花边、箭头主题)。
        -   ⚠️ 增强失败通知的视觉提示，使其更加醒目。
        -   ⚙️ 配置界面添加通知样式选择器。
    -   `v1.0.0`: 🎉 初始版本发布，提供基础的爱快路由备份功能。

### 🛡️ OpenWrt备份助手 (OpenWrtBackup)
-   **版本**: `v1.0.0`
-   **作者**: [@jinxi](https://github.com/xijin285)
-   **简介**: 自动备份您的 OpenWrt 路由器配置，通过 SSH 连接执行备份命令并下载备份文件，支持定时任务和备份文件管理。
-   **主要功能**:
    -   ⏰ **定时任务**: 根据设定的 CRON 表达式自动执行备份。
    -   ⚙️ **SSH 连接**: 通过 SSH 连接到 OpenWrt 设备执行备份操作。
    -   🔧 **自定义备份命令**: 允许用户配置在 OpenWrt 上执行的具体备份命令 (例如 `sysupgrade -b <path>`)。
        -   支持文件名占位符: `{timestamp}` (YYYYMMDDHHMMSS) 和 `{date}` (YYYY-MM-DD)。
        -   若命令中未指定占位符，则自动附加时间戳以确保文件名唯一。
    -   📂 **备份管理**:
        -   自定义本地备份文件存储路径。
        -   设置备份文件保留数量，自动清理旧备份。
        -   支持多种 OpenWrt 备份文件名格式的日期解析。
    -   🔔 **结果通知**: 备份成功或失败时发送通知。
        -   支持简洁和详细两种通知样式。
    -   ⚡ **即时执行**: 支持通过UI触发"立即运行一次"备份。
    -   📊 **历史记录**: 在插件页面查看详细的备份历史和状态。
    -   🔄 **智能重试**: 备份过程中的自动重试机制。
-   **依赖**: 需要目标 OpenWrt 设备支持 SSH，并且 MoviePilot 运行环境能够访问该设备。插件内部实现 SSH 操作需要 `paramiko` Python 库。
-   **初始版本 (`v1.0.0`)**:
    -   提供基础的 OpenWrt 备份功能，包括 SSH 连接配置、自定义备份命令、定时备份、备份管理和通知。
