# 115云盘签到插件

这是一个用于MoviePilot的115云盘自动签到插件。

## 功能特点

- 支持115云盘每日自动签到
- 简单易用的配置界面
- 签到结果通知

## 使用方法

1. 在MoviePilot中安装此插件
2. 在插件配置页面输入您的115云盘Cookie
3. 保存配置后插件将自动执行签到

## 获取Cookie方法

1. 登录115云盘网页版
2. 打开浏览器开发者工具（F12）
3. 在Network标签页中找到任意请求
4. 在请求头中找到Cookie字段
5. 复制完整的Cookie值

## 注意事项

- Cookie中包含您的登录凭证，请勿泄露给他人
- 建议定期更新Cookie以确保签到功能正常
- 如遇到签到失败，请检查Cookie是否过期
