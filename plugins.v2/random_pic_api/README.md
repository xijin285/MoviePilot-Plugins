# 随机图片API插件

这是一个为MoviePilot设计的随机图片API插件，支持自动识别设备类型并提供相应的图片。

## 功能特点

- 自动识别访问设备类型（移动端/PC端）
- 支持横屏和竖屏图片分类
- 支持多种图片格式（jpg, jpeg, png, gif, webp）
- 自动将图片转换为WebP格式以优化性能

## 安装说明

1. 将插件文件夹复制到MoviePilot的plugins.v2目录下
2. 在MoviePilot中启用插件
3. 创建以下目录结构：
   ```
   random_pic_api/
   ├── images/          # 原始图片目录（可选）
   ├── portrait/        # 竖屏图片目录
   └── landscape/       # 横屏图片目录
   ```

## API接口

- `/random` - 根据设备类型返回随机图片
- `/random/mobile` - 返回竖屏图片
- `/random/pc` - 返回横屏图片

## 图片处理

插件会自动处理`images`目录下的图片：
1. 检测图片方向（横屏/竖屏）
2. 将图片转换为WebP格式
3. 根据方向分类存储到对应目录

## 注意事项

- 建议使用WebP格式的图片以获得最佳性能
- 图片大小限制为178956970像素（约13000x13000）
- 确保图片目录具有适当的读写权限
